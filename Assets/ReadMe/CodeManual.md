# Code Manual

## Coding規約

### 命名規則
 - privateの変数は先頭アンダーバーのローワーキャメルケースで命名する。<br>
 　　例: _exampleVariable

 - publicの変数、プロパティはアッパーキャメルケースで命名する。<br>
 　　例: ExampleVariable

 - 定数(const)は全てアッパースネークケースで命名する。(ただし、自動生成コードは例外とする)<br>
     例: EXAMPLE_VARIABLE

 - メソッド、クラス、構造体、Enumはアッパーキャメルケースで命名する。<br>
     例: ExampleClass
